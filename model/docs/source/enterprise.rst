Enterprise
==========

Flower Labs works with organisations that want to evaluate, deploy, adapt, or
integrate Lizzy and future Flower models.

Work with us
------------

To request Lizzy Enterprise, please use this form to `contact Flower
<https://flowerlabs.typeform.com/to/oSxsEvgq>`_.

Enterprise work can include:

- confidential evaluations on domain-specific prompts, policies, and workflows
- deployment guidance for GPU serving, local inference, and quantized models
- model adaptation for tone, terminology, retrieval, and task performance
- safety, monitoring, and quality review for production use
- integration with existing applications, data systems, and governance
  processes

Typical engagements
-------------------

.. list-table::
   :header-rows: 1

   * - Need
     - Support
   * - Evaluate Lizzy for an internal use case
     - Benchmark design, test prompts, risk review, and deployment
       recommendations.
   * - Deploy Lizzy in production
     - Runtime selection, serving architecture, performance tuning, and
       operational review.
   * - Adapt the model to a domain
     - Data review, evaluation design, preference tuning, and release
       validation.
   * - Run local or confidential inference
     - GGUF, vLLM, and Transformers deployment guidance with appropriate
       monitoring and access controls.
